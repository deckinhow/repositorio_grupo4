## Branches
Branches são as "raízes" do nosso projeto. Cada branch têm um propósito.
Teremos duas branches, sendo elas a branch main e a develop.

### A branch "main" (a versão final):
É a nossa versão final, onde todas as criações e quaisquer alterações finais do código serão salvas. O código não será criado a partir desta branch, mas sim a partir da branch de desenvolvimento.

### A branch "develop" (o desenvolvimento):
É onde iremos desenvolver nosso código. Todas as criações, alterações e quaisquer outras coisas serão salvas aqui, para que revisemos antes de envia-las para a branch main.
