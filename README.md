# Repositório do Grupo 4

Aqui se encontram as principais informações e orientações sobre as nossas tarefas.


## Branches
Branches são as "raízes" do nosso projeto. Cada branch têm um propósito.
Teremos duas branches 'principais' e cinco outras branches.

### A branch "main" (a versão final):
É a nossa versão final, onde todas as criações e quaisquer alterações finais do código serão salvas. O código não será criado a partir desta branch, mas sim a partir da branch de desenvolvimento.

### A branch "develop" (o desenvolvimento):
É onde iremos desenvolver nosso código. Todas as criações, alterações e quaisquer outras coisas serão salvas aqui, para que revisemos antes de envia-las para a branch main. Também, não iremos trabalhar diretamente nesta branch, mas sim nas branches individuais.

### As branches individuais (as criações e mudanças):
É onde cada indivíduo irá realizar as suas tarefas e, após, envia-la para a branch develop. Cada indivíoduo deve criar a sua própria branch e trabalhar a partir dela.

Para criar uma branch, basta ir na parte "branch" e digitar o nome da sua branch (ex.: fulaninho) e criá-la.

Quando realizar alterações na sua branch, para salva-las na branch main você deve fazer um Pull Request.
